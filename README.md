## 1. Crop images using MTCNN

Keep the script mtcnn1.py in the root directory where the Driver Drowsiness DataSet folder is.
This script will read the images in the folder and create a new folder ***ProcessedImages*** with 2 sub-folders.

**Run:**  
`python3 mtcnn1.py`
